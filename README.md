
# Main

● Developed a website for content sharing with full CRUD functionality as well as search functionality using React and Django. Managed routing with React Router, integrated a REST API using Django REST framework, and connected to a PostgreSQL database. Supported up to 30 users and implemented search functionality.

● Implemented real-time search functionality with Elasticsearch for instant creator discovery.

● Improved website performance with Redis caching.

● Dockerized the project and deployed it on Railway using Railway CLI.
